# Executing C++ Code

- Using g++ compiler

    - Cpmpiling the code
        ```bash
        g++ -o <name of executable that has to be created> <name of the file to be compiled>
        ```
    - Running the executable
        ```bash
        ./<name of the executable>
        ```

- Using gcc compiler
    - Cpmpiling the code
        ```bash
        gcc <name of the file to be compiled> -lstdc++
        ```
    - Running the executable
        ```bash
        ./a.out
        ```
